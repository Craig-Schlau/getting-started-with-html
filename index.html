Important Concepts From "The Basics of the Web and HTML"

Major Components of the Web

The Browser - A program that runs on your computer to display files from the web.

The Internet - The world's largest computer network

HTTP - The main protocol of the Web

Servers - Computers that hosts the files that make up the Web.

HTML

HTML stands for Hypertext Markup Language. HTML is the main type of documentation on the web.

HTML is made up of:
Text Content (what you see)
Markup (what it looks like)
References to other documents (ex. images and videos)
Links to other pages

For more information on HTML, please click here.

Basic HTML Vocabulary

Tag - An HTML tag is always contained within angled brackets. Most tags have an opening bracket and a closing bracket. However, some tags (void tags) do not require a closing bracket.

Element - An HTML element refers to everything within a set of opening and closing tags.

Attribute - An attribute is a property of an HTML element. An example is using href to reference a URL.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.

Five Things to Take Away From This Lesson

1) What is a webpage?
A webpage is a text document written in a language called HTML. Web browsers read these documents and then interpret and display them.

2) How Coding Works
Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer wrote.

3) Computers are stupid.
Programmers need to write exactly the way a computer understands (also known as writing with correct "syntax").

4) Programmers can't remember everything.
There are too many details to remember everything. Programmers can always look up how to write some code. Google is your friend.
